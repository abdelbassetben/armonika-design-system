## States

### Disabled

When a radio button is disabled, it is faded and cannot be interacted with.

## Properties

| Property   | Type           | Default | Description                               |
| :--------- | :------------- | :------ | :---------------------------------------- |
| `size`     | `"xs" \| "sm"` | `"sm"`  | The size of the radio group item.         |
| `disabled` | `boolean`      | `false` | Whether the radio group item is disabled. |
