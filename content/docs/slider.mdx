An input where the user selects a value from within a given range. Built with [Base UI Slider](https://base-ui.com/react/components/slider), following the [shadcn/ui slider](https://ui.shadcn.com/docs/components/base/slider) patterns.

The **default** variant shows a label and min–max bounds above the track, with the current value under each thumb. The **special** variant is the plain slider for custom layouts.

## Properties

| Property  | Type                     | Default     | Description                                                                                    |
| :-------- | :----------------------- | :---------- | :--------------------------------------------------------------------------------------------- |
| `variant` | `"default" \| "special"` | `"default"` | Layout variant. `default` adds label, bounds, and thumb values. `special` is the plain slider. |
| `label`   | `React.ReactNode`        | —           | Label shown above the track (default variant only).                                            |
| `min`     | `number`                 | `0`         | Minimum value. Shown in the header span (default variant).                                     |
| `max`     | `number`                 | `100`       | Maximum value. Shown in the header span (default variant).                                     |

See the [Base UI Slider API](https://base-ui.com/react/components/slider) for all root props (`step`, `orientation`, `disabled`, `onValueChange`, etc.).
