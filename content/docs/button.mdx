## Properties

| Property | Type                                                                                                                                                 | Default     | Description                     |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | ------------------------------- |
| variant  | `"default"` \| `"outline"` \| `"secondary"` \| `"ghost"` \| `"destructive"` \| `"success"` \| `"link"`                                               | `"default"` | The variant style of the button |
| size     | `"default"` \| `"xs"` \| `"sm"` \| `"md"` \| `"lg"` \| `"xl"` \| `"icon"` \| `"icon-xs"` \| `"icon-sm"` \| `"icon-md"` \| `"icon-lg"` \| `"icon-xl"` | `"default"` | The size of the button          |
| disabled | boolean                                                                                                                                              | `false`     | Whether the button is disabled  |
| children | ReactNode                                                                                                                                            | -           | The content of the button       |
