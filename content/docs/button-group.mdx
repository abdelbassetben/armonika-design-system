## Properties

### ButtonGroup

| Property    | Type                           | Default        | Description                                                       |
| ----------- | ------------------------------ | -------------- | ----------------------------------------------------------------- |
| orientation | `"horizontal"` \| `"vertical"` | `"horizontal"` | The layout orientation of the grouped controls                    |
| spaced      | `boolean`                      | `false`        | Adds gap between the buttons instead of seamlessly combining them |
