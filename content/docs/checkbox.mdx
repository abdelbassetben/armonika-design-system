## States

### Disabled

When a checkbox is disabled, it is faded and cannot be interacted with.

## Properties

| Property   | Type                   | Default   | Description                       |
| :--------- | :--------------------- | :-------- | :-------------------------------- |
| `size`     | `"xs" \| "md" \| "lg"` | `"md"`    | The size of the checkbox.         |
| `icon`     | `"check" \| "minus"`   | `"check"` | The icon displayed when checked.  |
| `disabled` | `boolean`              | `false`   | Whether the checkbox is disabled. |
