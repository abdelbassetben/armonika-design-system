## Properties

| Property | Type                                                                                                                                                            | Default     | Description                               |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | ----------------------------------------- |
| variant  | `"default"` \| `"outline"` \| `"primary-flat"` \| `"gray"` \| `"green"` \| `"red"` \| `"orange"` \| `"lime"` \| `"emerald"` \| `"teal"` \| `"cyan"` \| `"info"` | `"default"` | The variant style of the badge            |
| size     | `"default"` \| `"sm"` \| `"md"` \| `"icon-only"`                                                                                                                | `"default"` | The size of the badge                     |
| radius   | `boolean`                                                                                                                                                       | `false`     | Whether the badge is fully rounded or not |
| children | `ReactNode`                                                                                                                                                     | -           | The content of the badge                  |
