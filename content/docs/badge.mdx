## Properties

| Property | Type                                                                                                                                                                                | Default     | Description                               |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------- | ----------------------------------------- |
| variant  | `"default"` \| `"outline"` \| `"secondary"` \| `"ghost"` \| `"destructive"` \| `"success"` \| `"green"` \| `"red"` \| `"orange"` \| `"emerald"` \| `"teal"` \| `"cyan"` \| `"link"` | `"default"` | The variant style of the badge            |
| size     | `"default"` \| `"sm"` \| `"md"` \| `"icon-only"`                                                                                                                                    | `"default"` | The size of the badge                     |
| radius   | `boolean`                                                                                                                                                                           | `true`      | Whether the badge is fully rounded or not |
| children | `ReactNode`                                                                                                                                                                         | -           | The content of the badge                  |
